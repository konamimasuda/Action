学習内容

LEARNING

独学では身に付きにくいデザインの基本から、Webサイトを制作する上での基礎技術、 実際にWeb現場で利用されている知識・技術まで学習します。 進化の早いWeb業界に合わせた、「今」必要な技術と知識をWSSで学習できます。

授業内容＆使用ツール

LESSON&TOOLS

各種プログラミング

Programing

Webサイトの構築に必要なHTML・CSS、Javascript・jQuery、RMD(レスポンシブウェブデザイン)、Sass(scss)・Gitなどの導入方法まで学習します。レスポンシブに対応したコーディングまでしっかりとお伝えします。

HTML

HTML

HTML(エイチティーエムエル)はWEBページを作成するための言語です。見出しや段落・表・リンクなど、文章が構成されている要素がどのような役割を持っているかを指示しています。HTMLの記述によって、検索エンジンがWEBページの構造を把握しやすくなります。よって、ほとんどのWEBページはHTMLとCSSという言語でその見た目が作られています。

CSS

CSS

CSS(シーエスエス)はWebページの文字の色や大きさ、背景、配置といった
スタイル（見た目）を設定する言語です。 一般的にWebサイトはHTML（エイチティーエムエル）という言語で記述されており、これにスタイルシートを記述する言語であるCSSを適用することで、私達が普段見ているWebサイトが構成されています。

JavaScript

JavaScript

JavaScript(ジャバスクリプト)はWebサイトに動きをつけるプログラミング言語です。
これを使用すると、ユーザーのアクションに応じたコンテンツの表示の他、地図やグラフィックアニメーションなども表示することができます。上記で説明したHTMLやCSSが翻訳された後で実行できます。

Sass

Sass

Sass(サス)は上記で説明したCSSを拡張して、書きやすく、見やすくしたものです。従来のCSSにデザイナーやプログラマーが抱いていた不満を解消したものといえます。SassはCSSの拡張言語で、効率的にCSSを記述するための言語であり、Webサービスを手掛ける多くの大企業が導入しています。

Git

Git

Git(ギット)とはファイルのバージョン管理が簡単にできるツールです。Gitで管理しているファイルであれば、コンピューター上でファイルの編集履歴を管理できるので、編集前のファイルを残したまま、新しいファイルを保存することができます。プログラマーにとって、多くのコードを編集した上で不具合が起きた時に元のバージョンへ戻すときがあるので、無駄な作業を省き、効率的なファイル管理を行えるのが、Gitの強みです。

デザイン・レイアウト

Design & Layout

デザインは「センス」が必要だと思われがちですが、デザインのセオリーを抑えておくことで誰でもデザイン力を向上・磨くことが可能です。基礎をしっかり学ぶことで、 より良いレイアウトを組むことができるようになります。

Photoshop

Photoshop

Photoshop(フォトショップ)はデザイン系写真編集ソフトです。写真や画像の加工・色の調整、複数画像の合成、テキストの追加や装飾などが自在に行えるツールです。写真の編集、デザインやイラストの作成などさまざまな用途に活用し、クリエイターや制作現場のプロから写真を楽しむライトユーザーまで幅広く使用されています。

XD

XD

XD(エックスディー)はデザイン系ツールです。WEBサイトやモバイルアプリなどの
画面デザインや試作品として動作を確認するプロトタイプ作成に使用されています。早く正確に高品質なデザインを作成することができ、まるで完成品を扱うかのようにプロトタイプを体感できるため、誰でも製品をイメージしやすいのが特徴です。

Figma

Figma

Figmaは(フィグマ)Webブラウザ上で簡単にデザインを作成できるツールです。リアルタイムに共同編集を行えます。ホームページやアプリの設計図であるワイヤーフレームや、試作品として動作を確認するプロトタイプを作成できます。また無料プランもあるので、手軽に使用できるのが魅力的な部分の一つです。

テキストエディタ

Text editor

コードを記述する際に使うソフトで、様々なエディタが存在しており、制作会社によって環境が違うのが特色です。よく使用されているのはMicrosoft Visual Studio Code、Sublimetext、Adobe Bracketsなどです。当校ではVisual Studio Code を使用します。

Visual Studio code

Visual Studio Code

VisualStudyCode(ビジュアルスタジオコード)は、Microsoft社の提供するHTMLやCSSなどの多くのプログラミング言語の編集を主な目的としたテキストエディタです。軽量で、機能が必要最低限に抑えられており、拡張機能が豊富なため、多くのプログラマーに使用されています。

その他ツール

Other tools

その他、授業を円滑に進めるためのコミュニケーションツールとして、ChatWorkを導入している他、Web業界では欠かせないCMS（コンテンツ・マネジメント・システム）の中で、日本で圧倒的なシェア率を誇るWord Pressを課外授業として行っております。

Chat Work

Chat Work

Chatwork(チャットワーク)は、国内で多くの企業に利用されているビジネスチャットツールです。国産のビジネスチャットであり、日本人のユーザーにとって使い方がわかりやすいことも多くの企業で利用されている理由のひとつです。素早いコミュニケーションを実現し、業務も効率化できます。

Quip

Quip

Quip(クイップ)は、データの管理やコミュニケーションのためのツールです。文書、スプレッドシート、スライドといったドキュメントを参加しているメンバー間でリアルタイムに作成・編集・共有できるオンラインツールで、さらにチャットができる機能が搭載されています。業務で必要となる文書などを、オンライン上で会話をしながら作成できるという点が大きな特徴です。

Word Press

CMS

CMS(シーエムエス)とは専門知識がない人も、Webサイトの作成・更新・運営ができるシステムです。その中でも特に人気の高いWordPress(ワードプレス)を使用します。理由は誰でも簡単に無料でWebサイトを作れることです。通常必要になるHTMLやCSSなどのコードによる編集を極力無くし、管理画面上でホームページ全体の投稿・加工・編集を可能にするシステムです。

半年間の流れ

CURRICLUM

01

基礎学習

1～2MONTH

BASIC LEARNING

Webサイト知識講習
Webサイト制作に必要なソフトや画像の仕組み、基本的なパソコンの使用方法など、 ６か月間学習する内容の基礎の部分を学習します。
HTML・CSS基礎実習
Webサイトを制作する時に使用するHTML(HTML Living Standard)や CSS(CSS3)など、基礎的な技術・知識をハンズオン形式で学習します。
Webデザイン基礎実習
Adobe Photoshop CCを使用し、基本的な操作や使い方、画像編集・写真加工を学習します。
Webデザイン制作実習
Webデザイン基礎実習で学んだことをベースに、デザインの基礎知識なども身に付けていきます。

1か月目の終わりには、習いたてのコードを使い、授業内容を取り入れた自己紹介サイトを作成し発表します。

02

応用学習

3～4MONTH

APPLIDE LEARNING

CSS設計
WSSでは、命名規則の一種であるBEM(ベム)を取り入れてコーディングが 効率的にできるようCSS設計を学びます。
コーポレートWebサイト実習
これまで学習した内容を用いて、グループワークを行います。 グループ毎に一つのWebサイトを企画からデザイン・コーディングまで手がけ、プレゼンを行います。
JS実習
JavaScript(jQuery)を利用してホームページに動く要素(スライドショーなど)や Webアプリケーションを追加する方法を、基本から応用まで実践的に制作しながら学習します。
Sass(SCSS)
Sass(SCSS)という拡張言語(メタ言語)の導入方法・基本的な使い方など学習します。 Sassとは、CSSを効率よくコーディングするための拡張言語です。

グループワークを通して、協調性が育まれ思考力・段取り力・対人力が養われます。

03

制作実習

5～6MONTH

PRODUCTION LEARNING

レスポンシブWebデザイン基礎実習
スマートフォンやタブレットなど、さまざまな端末に対応した マルチデバイスなWebサイト制作を学習します。
レスポンシブWebデザイン制作実習
就職活動に必須アイテムとなっているポートフォリオ(作品集)サイトを、 レスポンシブとなるよう(マルチデバイス対応)設計・制作します。
Webサイト企画・デザイン実習
実際にある企業またはお店のWebサイトを、模擬的な案件として制作します。 企画・デザインからコーディングまで行う事で、 仕事に近い間隔で流れを理解することができます。
Webサイト制作実習
Webサイト企画・デザイン実習内で制作したデザインカンプをもとにコーディング作業を行います。 6か月間の集大成でもありますので、訓練修了までに、完成させることを目指します。
