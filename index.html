What
I've
Learned

What I learned for 6 months and
how I'd like to utilize hereafter

半年間で学習したことや、持っているスキルをご紹介します。

  • HTML LS

    Web制作の基礎となる部分のため、コンテンツの意味をしっかり考えながらマークアップすることを心がけています。

  • CSS3

    誰にとっても見やすく、わかりやすい記述ができるように自分の中でルールを作り、それに基づいて書くといった工夫をしています。

  • Sass

    よく使う技術を部品化し、修正が必要な場合に手間をかけず対処できるような記述を心がけています。

  • SCSS

    SassでCSSを記述する際は、SCSS記法を使って書いています。ネストの構造を考えながら、見やすいCSSを書く努力をしています。

  • jQuery

    少ない記述で手軽に使えるため、簡単な動きをつけたい時に重宝しています。

  • JavaScript

    jQueryの書き換えや自作サイトにライブラリやプラグインを使用し、今後の仕事で使えるようにJSを勉強しています。

  • Git

    複数人での使用経験はありませんが、個人の制作物のバージョン管理をGithubで行なっているため、基本的な使用法は習得しています。

  • WordPress

    Wordpress履修後にここの説明を追加する。

  • Adobe Photoshop

    写真の色調補正や合成などの基本的な画像編集ができます。

  • Adobe XD

    ワイヤーフレームや、プロトタイプを用いたデザインカンプの作成ができます。

  • Figma

    XD同様、ワイヤーフレームやデザインカンプの作成が可能です。グループワーク時に、複数人でのプロジェクト編集経験があります。

  • Canva

    プレゼン資料の作成や、簡単な画像・動画の編集が可能です。

  • Microsoft Excel

    基本的な操作や、簡単な関数を使った計算などができます。

  • Microsoft Word

    基本的な操作や、簡単な文書を作成することができます。

  • Microsoft PowerPoint

    基本的な操作と、プレゼン資料などのスライドを作成することができます。
